# OWASP Juice Shop – Security Assessment & Hardening

This repository documents a complete 3-week security assessment and hardening project performed on the OWASP Juice Shop application.

## Covered Areas
- Web application security assessment
- Vulnerability identification
- Secure coding implementation
- Penetration testing
- Logging and monitoring
- Final security checklist

⚠️ OWASP Juice Shop is an intentionally vulnerable application used strictly for educational purposes.
